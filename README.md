# StackFolio

StackFolio is an AI-assisted developer portfolio builder designed to help students and professionals create structured, recruiter-ready portfolios in minutes.

## Vision

Build the fastest and simplest portfolio platform for developers — starting with Africa and scaling globally.

## Core Features

Authentication system
Portfolio editor
Structured section engine
Template switching
AI summary generator
GitHub repository import
Recruiter-mode portfolio view
Static export support

## Tech Stack

Next.js
Supabase
Vercel
Gemini AI API

Phase: MVP Development

We are currently building the first working release.
